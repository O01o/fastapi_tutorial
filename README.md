# python_parakeet

## 機能要件

音声ファイルをポストすることで、JVSコーパスを使って学習されたCoquiTTSにデータが投げられ、インコボイスへと変換することができる技術。

## 開発環境
### 開発言語
Python
### 必要なパッケージ
pip install fastapi

pip install uvicorn　　

pip install gunicorn

### 実行コマンド
uvicorn main:app
